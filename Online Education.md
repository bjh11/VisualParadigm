1) 온라인 강의 플랫폼(Online Education) 업무 정의서

1. 액터 정의
   - 학생(Student): 강의를 검색하고 수강하며, 학습 활동을 수행하는 최종 사용자 / 강의 수강, 과제 제출 등
   - 강사(Instructor): 강의 콘텐츠를 생성하고, 수강생의 학습 활동을 관리하 및 평가하는 사용자 / 콘텐츠 업로드, 진도 모니터링 등
   - 관리자(Administrator): 플랫폼의 전반적인 시스템 운영, 설정 및 사용자 계정을 관리하는 운영자 / 계정 관리, 시스템 설정 구성 등

2. 주요 유스케이스 상세 정의
   - 강의 검색 / 학생 / 학생이 키워드, 카테고리, 강사명 등 다양한 조건으로 시스템에 등록된 강의를 탐색하고 목록을 확인하는 기능
   - 강의 수강 등록 / 학생 / 학생이 수강을 희망하는 강의를 자신의 수강 목록에 추가하는 기능
   - 강의 시청/수강 / 학생 / 등록한 강의의 콘텐츠(영상, 자료)를 시스템을 통해 시청&열람하는 기능
   - 강의/콘텐츠 업로드 / 강사 / 강사가 강의 영상, 교안, 실습 파일 등 강의 운영에 필요한 모든 콘텐츠를 플랫폼에 등록하고 관리하는 기능
   - 진도/과제 채점 / 강사 / 학생이 제출한 과제나 학습 결과물을 검토하고 평가하여 점수를 부여하며, 피드백을 기록하는 기능
   - 사용자 계정 관리 / 관리자 / 관리자가 학생, 강사 계정을 생성, 수정, 활성화/비활성화하거나 삭제하는 기능
   - 시스템 설정 구성 / 관리자 / 강의 카테고리, 수강료 정책, 서버 설정 등 플랫폼 전반의 운영 환경을 설정하고 변경하는 기능
